🌦 Animated Weather App

A modern, interactive, and fully animated Weather Application built using HTML, CSS, and JavaScript with real-time data from the OpenWeatherMap API. This project combines beautiful UI design with dynamic weather animations and a smooth user experience.

✨ Features

🔍 Search Any City – Get instant weather updates for any location

🌡 Real-Time Temperature Display

🤗 Feels Like Temperature

📈 Max & Min Temperature

🕒 Live Digital Clock (AM/PM)

🌅 Day Status Detection – Morning / Afternoon / Evening / Night

🌙 Dark & Light Mode Toggle

📅 5-Day Weather Forecast

🎥 Background Video Integration

☀️🌧❄️ Animated Weather Effects:

🔹Rotating Sun

🔹Moving Clouds

🔹Falling Rain

🔹Snowflakes

🔹Thunder & Lightning

🔹Moon & Twinkling Stars

💎 Glassmorphism UI Design

📱 Responsive & User-Friendly Layout

🛠 Tech Stack

🔹HTML5 – Structure

🔹CSS3 – Styling, Animations, Glassmorphism

🔹JavaScript (ES6) – Logic, API Handling, DOM Manipulation

🔹OpenWeatherMap API – Real-time Weather Data
