# 🛒 E-Commerce Web Application

A fully functional and interactive **E-Commerce Web Application** built using **HTML, CSS, and Vanilla JavaScript**.  
The project demonstrates real-world frontend development concepts such as **API integration**, **dynamic UI rendering**, and **Local Storage based cart management**.

This application allows users to browse products, add them to the cart, manage quantities, and retain cart data even after page refresh.

---
Let's try
⭐https://e-commerceeweb.netlify.app/

## 🎥 Project Overview (As Shown in Demo Video)

- Product list loads dynamically on page load
- Products are fetched using an external API
- Each product card displays image, title, price, and action buttons
- Users can add products to cart
- Cart items persist using browser Local Storage
- Cart automatically updates quantity and total price
- Smooth and user-friendly shopping experience

---

## ✨ Key Features

- 🛍️ Dynamic product listing using REST API
- ➕ Add to Cart functionality
- 🔄 Increase / Decrease product quantity
- 🗑️ Remove product from cart
- 💾 Cart persistence using Local Storage
- 📱 Responsive UI for all screen sizes
- ⚡ Fast and lightweight (No framework used)

---

## 🛠️ Technologies Used

- **HTML5** – Semantic structure  
- **CSS3** – Styling & responsive layout  
- **JavaScript (ES6)** – Logic, DOM manipulation  
- **REST API** – Fetching product data  
- **Local Storage** – Cart data persistence  
- **Git & GitHub** – Version control & deployment
