🛒 Product CRUD App (JavaScript Practical Exam)

A simple and interactive Product Management Web Application built using HTML, CSS, JavaScript, Bootstrap, and LocalStorage.
This project demonstrates complete CRUD operations along with sorting, searching, filtering, and a clean animated UI.

📌 Project Objective

The objective of this project is to create a product page that allows users to:

Add new products<br>

View product list dynamically<br>

Edit existing products<br>

Delete products<br>

Store data permanently using LocalStorage<br>

This project provides hands-on experience in JavaScript DOM manipulation and web storage.

🚀 Features

✅ Create, Read, Update, Delete (CRUD)<br>

💾 LocalStorage data persistence<br>

🔍 Search products by title<br>

🔃 Sort products by price (Low → High / High → Low)<br>

🗂️ Filter products by category<br>

🎨 Bootstrap-based responsive UI<br>

✨ Smooth animations & hover effects<br>

🧹 Auto-clear input fields after add/edit<br>

🧑‍💻 Technologies Used

HTML5

CSS3

JavaScript (ES6)

Bootstrap 5

Browser LocalStorage

⚙️ How It Works
➕ Add Product

User enters product title, price, image URL, and category

Validation ensures title and price are required

Product is saved with a unique ID

👀 Display Products

Products are displayed dynamically in a table

Image, title, price, and category shown clearly

✏️ Edit Product

Clicking Edit loads data into the form

Changes are updated using product ID

❌ Delete Product

Product is removed instantly

UI updates smoothly with animation

💾 LocalStorage

Data remains saved even after page refresh

Uses localStorage.setItem() and getItem()
