# 🔍 Input Value Display App

A simple web app that takes user input and displays it using JavaScript.
Perfect for beginners to understand **DOM manipulation** and **event handling**.

## ⚙️ Features

* ✍️ User can enter text
* 🔘 Button click event
* 📢 Displays input value
* 🎯 Beginner-friendly project

---

## 💻 How It Works

1. User enters text in the input field
2. Clicks the **Find** button
3. JavaScript captures the input value
4. The value is displayed using alert or on the screen
