# 🔢 Counter App

A simple and interactive **Counter Application** built using **HTML, CSS, and JavaScript**.
Perform basic arithmetic operations with a clean and modern UI.

## ⚙️ Features

* ➕ Increment (`++`)
* ➖ Decrement (`--`)
* ✖ Multiply by 2 (`*=2`)
* ➗ Divide by 2 (`/=2`)
* 🔢 Modulus (`%=2`)
* 🔄 Reset Counter
* 🎨 Modern UI with animated buttons

---

## 💻 How It Works

* The counter value is displayed at the top
* Buttons perform arithmetic operations on click
* JavaScript updates the value dynamically using DOM manipulation

---

## 📌 Code Example

```javascript
const count = document.getElementById("count");

button1.addEventListener("click", () => {
    count.textContent = Number(count.textContent) + 1;
});
```
